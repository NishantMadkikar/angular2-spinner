# Angular2 Spinner

Simple preloading indicator.

![preview](preview.png)

## Instalation

```
import { SpinnerModule } from 'angular2-spinner';
....
imports: [
    SpinnerModule,
],
...
 ```

## Usage

**Import:**

`import { SpinnerComponent } from "angular2-spinner";`

**Add component:**

`<spinner [size]="20" [tickness]="1" [color]="#ff66ff"></spinner>`
